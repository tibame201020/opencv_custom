# 產品情境

## 專案目的
建立一個現代化的 Web 前端介面，用於控制和監控 OpenCV 自動化腳本的執行。

## 問題解決
1. **腳本管理**：提供直觀的介面來瀏覽、選擇和執行不同的自動化腳本
2. **參數設定**：動態渲染不同腳本所需的參數配置介面
3. **執行監控**：即時顯示腳本執行狀態和日誌輸出
4. **多腳本支援**：同時管理多個腳本的執行，每個腳本獨立分頁

## 使用者體驗目標
1. **簡潔直觀**：清晰的雙 Tab 結構（Execution / Setting）
2. **即時反饋**：執行狀態、日誌即時更新
3. **個人化**：支援主題切換和語系選擇
4. **響應式設計**：適配不同裝置和螢幕尺寸

## 核心功能
### Execution Tab
- 左側 Drawer：腳本清單，支援平台篩選和關鍵字搜尋
- 右側 Tabs：多腳本分頁管理
  - Control Block：參數設定和執行控制
  - Log Block：即時日誌顯示和搜尋

### Setting Tab
- Theme Selector：30+ DaisyUI 主題選擇
- Locale Selector：繁體中文 / English 切換
